Pre-read. The design is locked, so this is a pose survey. Fresh-context crop audits rate all 14 renders (7 poses × both models) as viable pose candidates — correct (non-mirrored) chirality on every one, both feet grounded, and the locked Kit-A design present (breastplate + fur mantle, plain cold-iron scimitar + plain steel shield, a bold chest sun, ankh pendant, clean-shaven, uncovered flame hair). Basing win: P3 advancing's R2 lifted-heel defect is fixed on both models. So pose is the decision; every candidate is buildable.

Blind-detector cross-check — real refinements, read the chips. Each card carries a non-overriding blind cross-check chip. The unbiased detector (describes the render without the spec, then compares it to that render's own pose prompt) surfaces genuine items the primed crop pass under-weighted — the R1 lesson repeating: (1) "one sun" creep — small sun-medallion cape-shoulder clasps show up on several renders (clearest on P5 riverflow25); the spec wants the chest sun to be the ONLY sun, so those clasps want to become plain studs. (2) P5 dawn-salute (riverflow25): the scimitar tip floats above the ground instead of resting on it — the intended grounded third-contact isn't achieved on that render (the gptimage2 P5 holds it better). (3) design-to-survive — the ankh+sun pendant reads as a plain ankh (the fused sun-disc isn't distinct), and hair tips / scimitar blade read thin; thicken these at the clay/turnaround/mesh stage. (4) build — several renders read a touch stout to the unbiased eye despite the lithe/fey brief; worth your eye on which pose+model best holds the fey read. Chirality is genuinely correct — an earlier blind pass run against the baseline spec false-flagged mirrors board-wide; re-running each render against its own pose prompt removed that frame artifact (logged in verdict-overturns), leaving the genuine kernel above. None of this blocks a pose pick — it's the R4 refinement + eye-check list. Raw blind pass: notes/r3-blind-detect.jsonl.
