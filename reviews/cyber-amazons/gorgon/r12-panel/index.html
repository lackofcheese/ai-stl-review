Gorgon — Round 12 (--no-remesh re-run)

📖 Round runbook →

Round 12 regenerates the two Round-11 keepers (non-clay gold + r2clay) at Meshy's raw highest-precision --no-remesh setting, instead of the remesh→300k default r10/r11 ran by mistake. Same crops, only the Meshy flag changes — so any quality change is the setting.

Pre-read — the fix worked, decisively. The non-clay gold --no-remesh mesh is now FULLY WATERTIGHT (0 non-manifold edges, 0 holes, single shell) vs the r11 remeshed version's 10 non-manifold edges, and its surface roughness dropped −56% (high-pass RMS 14.8 µm vs 33.9). r2clay --no-remesh: 1 non-manifold edge (vs 18), same −56% roughness. This confirms the r11 gold "surface micro-noise" was the remesh decimation artifact, not the photoreal input — the non-clay path produces a clean, watertight print master. Meshes are 37 mm, ~1.0 cm³, 455–474k faces (raw high-poly). Matches the Igor benchmark (−62% RMS, non-manifold 1 vs 20).
